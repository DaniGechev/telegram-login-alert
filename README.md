# Telegram Login Alert Project

This is a Node.js Express web application that implements authentication and sends a Telegram notification when a user logs in.

## Features
- Login system (username & password)
- Session authentication
- IP address tracking
- Telegram login alert

## How to run

1. Install dependencies:
   npm install

2. Create a .env file with:
   BOT_TOKEN=your_token
   CHAT_ID=your_chat_id
   PORT=3000

3. Start the server:
   node index.js

4. Open in browser:
   http://localhost:3000
